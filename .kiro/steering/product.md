# Product Overview

Cloud Solution Library is ANWB's comprehensive documentation platform for AWS account setup and cloud solution management. It provides standardized guidance for:

- AWS account provisioning and setup workflows
- Service Catalog products (GitLab OIDC integration, CDK Bootstrap)
- Access management and security best practices
- Infrastructure deployment patterns

The documentation is built with MkDocs and automatically published to GitLab Pages and the Engineering Portal (Backstage) via TechDocs.

## Key Components

- **Documentation Site**: MkDocs-based documentation with Material theme
- **Service Catalog Templates**: CloudFormation templates for standardized AWS resources
- **TechDocs Publisher**: CI/CD pipeline and Lambda-based ingestion system for Backstage integration
- **Vale Linting**: Custom style rules for documentation quality (EnEn style guide)

## Target Audience

Cloud Center of Enablement (CCoE) team and ANWB development teams setting up AWS infrastructure.
